Bizstrap

Rapid Business Applications Development Framework based on Apache OFBiz

View project on GitHub

Welcome to Friends of Apache OFBiz project Pages.

BizStrap is a light weight java application development framework that ties together the goodness of Apache OFBiz with a ease of Twitter Bootstrap:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you are new to Apache OFBiz follow the link if you are new to Twitter Bootstrap find out more here

Bootstrap ready Screen Widgets

Liked the rapid UI development of OFBiz but struggled with the User Interface. Well BizStrap provides the same Screen, Form, Menu and related widgets but now rendering Bootstrap ready HTML.

No Modules?

That's right. We have been developing OFBiz based applications for a while since we liked the simplicity. However, were always pained buy the interlinked modules many of which we didn't really need. So, we made an attempt to retain the bare bones and strip off everything else so that you can write your own application without the unwanted baggage of modules that hang around only to satisfy the dependency.
